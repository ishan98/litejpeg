Bases: litex.soc.interconnect.stream.PipelinedActor, litex.gen.fhdl.module.Module

This module will connect the Rle core datapath with the input and output either from other modules or from the Test Benches. The input is been taken from the sink and source and is been transferred to the RLE core datapath by using read and write count.

class litejpeg.core.rle.entropycoder.EntropyDatapath(i, *args, **kwargs)¶

Bases: litejpeg.core.rle.entropycoder.EntropyDatapath

It contains the steps for the Entropycoder to calculate the bits for the Amplitude to store.

sink : Get the input from the other modules.
The input is the modulus of the input.

source : Give the output to the other modules. input_data : modulus of input of the matrix. get_data : Contains the temporary data for calculating the size. size : The number of bits required for storing th amplitude.
