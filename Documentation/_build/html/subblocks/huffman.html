class litejpeg.core.huffman.huffmancore.HuffmanDatapath(i, *args, **kwargs)¶

Bases: litejpeg.core.huffman.huffmancore.HuffmanDatapath

This gives the datapath for the serialization of the input data. The data coming to the module is been controlled by using ready_data_write and ready_data_read, as when the module is ready then the input ready_data_write will be 1 and when the data to the output is ready then ready_data_read is 1.

Amplitude : 12 bits
Non-zero amplitudes in the matrix.
Size : 4 bits
Number of bits required to store the amplitude.
Runlength : 4 bits
Number of zeros before the non-zero amplitude.
word_count: 6 bits
Display the index of the matrix.
source : 8 bits
Output of the serial data coming out.

class litejpeg.core.huffman.huffmancore.HuffmanEncoder¶

Bases: litex.soc.interconnect.stream.PipelinedActor, litex.gen.fhdl.module.Module

This Module defines the main module for getting the input and output to the HuffmanDatapath from the other module. The input is been taken from the ‘sink’ with the Endstream of length 20 bits and the output is been sent by using ‘source’ with 9 bits. The input and the output are been synchronized by using the ready_data_read and ready_data_write.

Used to build Huffman Tables
