Zigzag Module¶

class litejpeg.core.zigzag.ZigZag¶

Bases: litex.gen.fhdl.module.Module

This will take the values for the zigzag module.

The values are been taken by in the form of serial input and stored in the memory named data_mem. Than another zigzag_mem is present which contains the order in which the values need to be rearranged. The data when read from the memory are than taken in order for zigzag_mem hence read in that form.

source = None¶

zigzag_mem : 64 blocks Stores the data in the form in which the data needs to be rearranged.

zigzag_read_port : Read the signal from the memory location as per
the address in the zigzag_mem.adr .
