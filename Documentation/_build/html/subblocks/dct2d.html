Qunatizer Module¶

class litejpeg.core.new_dct6.DCT(dw=12, dct_block=64)¶

Bases: litex.soc.interconnect.stream.PipelinedActor, litex.gen.fhdl.module.Module

This will get the input into the DCT module in the form of 64 blocks with 8 bits each. The input is taken in the form of serial input. After obtaining all the 64 inputs from the serial input they are than pass to the DCT module. Also after obtaining the 64 blocks from the DCT module, these are transferred to the output in the form of serial data.

class litejpeg.core.new_dct6.DCTDatapath(i, *args, **kwargs)¶

Bases: litejpeg.core.new_dct6.DCTDatapath

This will take the input as YCbCr matrix and than convert it into the DCT matrix. The matrix so formed is been passed through the zigzag or the quantization module so that the compression done will be maximum.

It takes the input in the form of 64 blocks of 8 bits each as unsigned int.

Gives 64 blocks of 8 bits with signed int.
